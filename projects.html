StudyVerse – AI-Powered Learning Platform

StudyVerse is an AI-powered full-stack learning platform that provides personalized educational content by generating notes and quizzes for each topic using Google Gemini API, managed securely through an admin-controlled system.

Key Responsibilities & Features

  • Designed and implemented a normalized MySQL database schema for users, courses, topics, quizzes, and progress tracking.
  • Developed backend logic and REST API endpoints using PHP.
  • Built a secure authentication system with role-based access control for admins and learners.
  • Integrated Google Gemini API to automatically generate AI-based notes and quizzes for each topic.
  • Implemented an admin panel to manage Gemini API key, course content, and AI-generated materials.
  • Created a responsive frontend interface for browsing courses, accessing notes, and attempting quizzes.
  • Implemented learning progress tracking and quiz performance monitoring.
  • Focused on scalability, usability, and data security.

Tech Stack

PHP, MySQL, JavaScript, HTML5, CSS3, Google Gemini API

Plag AI App – Text Plagiarism Detection

Plag AI App is a machine learning-powered text plagiarism detection application with an intuitive user interface for analyzing content similarity and generating detailed reports.

Key Responsibilities & Features

  • Designed and developed a clean and intuitive user interface for text plagiarism detection.
  • Integrated Machine Learning-based text comparison logic with front-end display components.
  • Improved usability by presenting clear similarity reports and root cause analysis visual feedback.
  • Implemented real-time text analysis and similarity scoring algorithms.
  • Created comprehensive visualization tools for plagiarism detection results.
  • Developed responsive design for seamless cross-platform compatibility.
  • Focused on accuracy, performance, and user experience in content analysis.

Tech Stack

Python, Machine Learning, Front-End UI Integration, Text Analysis
